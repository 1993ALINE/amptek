AMPTEK ENGINEERING — HOW TO RUN (Windows)
==========================================

Open the site from the services folder (inside your amptek project).

1. Get latest code (PowerShell):
   cd C:\Users\Administrator\desktop\amptek
   git pull origin cursor/bdshop-style-store-c96e

2. Go into the services folder:
   cd shop

3. Double-click:  START-HERE.bat

4. Your browser will open:  http://localhost:8080/index.html

   (If Node.js is not installed, install from https://nodejs.org then run START-HERE.bat again)

Pages:
  index.html    — Home / services
  cart.html     — Quote list
  product.html?id=1  — Service detail (click any card)
  contact.html  — Contact form & phone/email

Do NOT rely on double-clicking index.html alone — use START-HERE.bat for full quote list, search, and detail pages.
