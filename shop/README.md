# Amptek Store — E-commerce Demo

BDShop-style storefront for **Amptek**: category hubs, product grids, BDT pricing, cart, and FAQ.

**Contact:** 01671-113615 · info@amptekeng.com · www.amptekeng.com

## Preview locally

Double-click `index.html`, or:

```bash
cd shop
python -m http.server 8080
# http://localhost:8080/index.html
```

## Features

- Category hub banners (Solar, Electronics, Gaming, Mobile)
- Most Sold / Most Discount / New Arrivals
- Shop by category, search, cart, product pages
- FAQ, COD/bKash messaging, nationwide delivery
